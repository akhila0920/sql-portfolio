## Overview

This repository serves as a resource for SQL practitioners, offering a variety of case studies, platform solutions, projects, and takehome challenges. These resources are designed to enhance SQL skills through practical applications and complex problem-solving scenarios across different business domains.

## Case Studies - 8 week SQL challenge

The 8 Week SQL Challenge by Danny Ma is a series of case studies designed to improve SQL skills. Each case study presents a real-world business scenario, requiring a series of questions to be answered using SQL queries. 

This repository contains solutions to these challenges, along with explanations and documentation of the methodologies used. The solutions for each case study are provided in their respective folders.
